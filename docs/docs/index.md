# Prime

Prime is a Python framework to perturbatively solve the infamous equations
called the gravitational closure equations. This system of equations allows
the determination of any gravitational Lagrangian to a given matter theory.

## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs help` - Print this help message.
